# bingobingo

## Backend

1. Create environment
   ```
   $ virtualenv venv
   $ source venv/bin/activate
   ```
2. Install required packages   
   ```
   $ cd backend
   $ pip3 install -r requirements.txt
   ```  
3. Execute server
   ```
   $ flask run
   ```

## Frontend

1. Install package
   ```
   npm install create-react-app
   ```
2. Install react package(only for the first time)
   ```
   cd frontend
   yarn install
   ```
3. Getting started
   ```
   yarn start
   ```
